# 오늘의 건강 — 기획 설계 스펙

**작성일**: 2026-04-17
**작성자**: PM (BEA)
**상태**: 승인 완료 → 디자인팀 핸드오프 대기

---

## 1. 서비스 개요

오늘의 건강은 건강 습관과 라이프스타일 정보를 비전문가 눈높이로 전달하는 **뉴스레터 기반 콘텐츠 사이트**다. AI 초안 + PM 편집 방식으로 매주 목요일 발행하며, 웹 아카이브를 통해 콘텐츠 미리보기 → 구독 전환 구조를 갖는다.

- **발행 주기**: 주 1회 (매주 목요일)
- **콘텐츠 구성**: 메인 아티클 1개 + 짧은 팁 2~3개
- **발송 도구**: 스티비(Stibee)
- **타겟**: 건강 정보가 필요한 일반 소비자/환자

---

## 2. 페이지 구조 (Information Architecture)

```
오늘의 건강
├── 메인 (/)
│   ├── 히어로: 서비스 한 줄 소개 + 구독 CTA
│   ├── 최신 뉴스레터 3개 미리보기 카드
│   └── 카테고리 탐색 링크
│
├── 아카이브 (/archive)
│   ├── 전체 뉴스레터 목록 (카드형)
│   ├── 카테고리/태그 필터
│   └── 각 카드: 제목 + 발행일 + 첫 2~3줄 미리보기
│
├── 아티클 상세 (/archive/[slug])
│   ├── 제목 + 발행일 + 카테고리
│   ├── 본문 앞부분 공개 (메인 아티클 300자 + 팁 1개)
│   ├── [구독 유도 배너] ← 컷오프
│   ├── 나머지 본문 블러/잠금 처리
│   └── 댓글/반응 영역
│
└── 구독 페이지 (/subscribe)
    ├── 이름 + 이메일 입력 폼 (스티비 연동)
    ├── 발행 주기·콘텐츠 소개 한 줄
    └── 구독 완료 후 → 감사 메시지 + 최신 아티클 안내
```

## 3. 핵심 기능 명세

### 3-1. 콘텐츠 구조 (뉴스레터 1개 단위)

| 구성 요소 | 분량 | 비고 |
|---------|------|------|
| 메인 아티클 제목 | 1줄 | |
| 메인 아티클 한 줄 요약 | 1문장 | |
| 메인 아티클 본문 | 600~900자 | 비전문가 눈높이 |
| 짧은 팁 | 100~150자 × 2~3개 | |

### 3-2. 카테고리 (초기 4개)

- 식단·영양
- 운동·활동
- 수면·회복
- 멘탈·스트레스

### 3-3. 구독 플로우

```
이름 + 이메일 입력
→ 스티비 구독 확인 메일 발송 (더블 옵트인)
→ 사용자 확인 클릭
→ 구독 완료 + 웰컴 메일 자동 발송
→ 매주 목요일 뉴스레터 수신
```

### 3-4. 미리보기 컷오프 기준

- 메인 아티클: 앞 300자 공개
- 팁: 1개만 공개, 나머지 잠금
- 잠금 이후: "전체 읽기 → 구독하기" 버튼 노출

### 3-5. 댓글/반응

| 기능 | 접근 권한 |
|------|---------|
| 이모지 반응 (👍 ❤️ 🤔) | 누구나 |
| 텍스트 댓글 | 구독자만 |

## 4. 콘텐츠 운영 방식

### 주간 발행 사이클

| 일정 | 작업 |
|------|------|
| D-3 월요일 | AI로 주제 초안 생성 (메인 아티클 + 팁 3개) |
| D-2 화요일 | PM 검토·편집 (비전문가 눈높이 확인, 사실 검증, 톤 다듬기) |
| D-1 수요일 | 스티비 발행 세팅 (제목/미리보기 문구, 카테고리/태그, 발송 예약) |
| D-day 목요일 | 뉴스레터 발송 + 웹 아티클 동시 게시 |

### AI 활용 원칙

- 출처 명시 요청 (근거 있는 정보만)
- "30대 직장인도 이해할 수 있게" 톤 지정
- 의학적 조언 아닌 정보 제공임을 명시

### 콘텐츠 URL 구조

- 슬러그 기반 영구 URL: `/archive/2026-04-17-수면-루틴`
- 스티비에 발송 원본 보관

## 5. 사용자 시나리오

### 시나리오 A: 신규 방문자 → 구독 전환

```
1. SNS/검색으로 아티클 링크 유입
2. 아티클 상세 페이지 접속
3. 메인 아티클 앞부분 읽음 (300자)
4. 컷오프 배너 노출 → "구독하고 전체 읽기" 클릭
5. 구독 페이지로 이동
6. 이름 + 이메일 입력 → 확인 메일 클릭
7. 웰컴 메일 수신 + 최신 아티클 전체 열람 가능
```

### 시나리오 B: 구독자 → 재방문·충성 독자

```
1. 매주 목요일 뉴스레터 수신
2. 흥미로운 제목 → 메일에서 클릭
3. 웹 아티클 전체 읽기
4. 이모지 반응 남기기
5. 아카이브에서 지난 글 탐색
6. 관련 카테고리 필터로 더 읽기
```

## 6. MVP 범위 확정

### MVP 포함 (1개월 내 출시)

|------|------|
| 메인 페이지 | 히어로 + 최신 3개 미리보기 + 구독 CTA |
| 아카이브 페이지 | 전체 목록 + 카테고리 필터 |
| 아티클 상세 페이지 | 미리보기 컷오프 + 구독 유도 |
| 구독 페이지 | 이름+이메일 폼 + 스티비 연동 |

| 텍스트 댓글 | 구독자만 |
| 카테고리 4개 | 식단·영양 / 운동·활동 / 수면·회복 / 멘탈·스트레스 |
