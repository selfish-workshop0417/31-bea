# 오늘의 건강 — PRD

## 1. 서비스 개요
오늘의 건강은 건강 습관과 라이프스타일 정보를 비전문가 눈높이로 전달하는 뉴스레터 기반 콘텐츠 사이트입니다. 복잡한 의료 정보를 누구나 이해할 수 있는 언어로 풀어내어, 일반 소비자와 환자가 일상에서 건강을 쉽게 챙길 수 있도록 돕습니다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning)
- 뉴스레터 정기 발송 (건강 습관·라이프스타일)
- 쉬운 언어로 의료/건강 정보 큐레이션
- MVP 범위: 구독 신청 페이지 + 첫 뉴스레터 발송

상세는 `planning/` 폴더 참조. [작성 예정]

## 4. 디자인 (Design)
친근하고 신뢰감 있는 톤앤매너, 읽기 편한 뉴스레터 레이아웃 중심
메인 → 구독 신청 → 뉴스레터 수신의 심플한 흐름

상세는 `design/` 폴더 참조. [작성 예정]

## 5. CX (Customer Experience)
건강 정보가 필요하지만 전문 용어가 어렵게 느껴지는 일반 소비자/환자
인지 → 구독 신청 → 첫 뉴스레터 수신 → 재방문의 여정

상세는 `cx/` 폴더 참조. [작성 예정]

## 6. 마케팅 (Marketing)
건강 관심 커뮤니티와 SNS를 통해 초기 구독자 확보
주요 채널: 인스타그램, 네이버 카페/블로그

상세는 `marketing/` 폴더 참조. [작성 예정]

## 7. 전략 (Strategy)
수익 모델 미정 — 구독자 수 확보 후 광고/스폰서십 또는 유료 구독 전환 검토
1개월 내 MVP 출시 후 피드백 기반 성장

상세는 `strategy/` 폴더 참조. [작성 예정]

## 8. 역할 배정 (가상)

| 팀 | 담당 영역 | 작업 폴더 |
|----|---------|----------|
| PM (본인) | 전체 기획·의사결정 | 루트 |
| 기획 팀 | 기능 정의, MVP 범위 | planning/ |
| 디자인 팀 | UI/UX, 비주얼 | design/ |
| CX 팀 | 페르소나, 여정맵, FAQ | cx/ |
| 마케팅 팀 | 런칭, 채널 | marketing/ |
| 전략 팀 | 수익 모델, 성장 | strategy/ |
