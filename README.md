# 오늘의 건강

비전문가 눈높이로 전하는 일상 건강 습관 뉴스레터

## 프로젝트 정보
- **유형**: 콘텐츠 사이트 (뉴스레터 중심)
- **타겟 사용자**: 건강 정보가 필요한 일반 소비자/환자
- **출시 목표**: 1개월 이내 MVP

## 핵심 기능
- 뉴스레터 발송 (건강 습관·라이프스타일 주제)
- 비전문가도 이해할 수 있는 쉬운 언어로 의료 정보 제공

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
